# Emotional Experience

## Core Emotion

When a golfer opens ParMind, their first feeling should be:

> "I've got this."

ParMind exists to replace uncertainty with confidence.

It should never create pressure or excitement for its own sake.

Instead, it should reinforce calm, preparation, and trust in the golfer's decision-making process.

## Design Implications

Every interaction should help the golfer feel:

- Prepared before the shot.
- Focused during the round.
- Reflective after the round.
- Optimistic about improvement.

If an interaction increases stress, distraction, or unnecessary excitement, it should be redesigned.
